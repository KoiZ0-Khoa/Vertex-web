# Vertex Web - Frontend

AI-powered web application for automatic study plan generation.

This repository contains the **Frontend (Client Application)** of the Vertex Web system.

---

## 🚀 Project Overview

Vertex Web is a full-stack study planning platform designed to help students:

- Automatically generate intelligent study plans
- Distribute tasks for group assignments
- Track deadlines effectively
- Receive AI-powered suggestions

This repository handles all **UI rendering, client-side state management, and API communication**.
